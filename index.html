Saucy Steve the Skateboarding Sloth

Score: 0

Tickets: 0

State: Hub

Ride: Skateboard

Combo: None

Skate the city and rack up points to earn tickets.
Controls
  • Move: WASD or Arrow keys (supports 8-direction)
  • Push for speed: K
  • Ollie/Jump: J
  • Spin: L (best while airborne)
  • Switch ride: Z (previous), X (next)
  • Enter competition zone: E
  • Return to hub (in competition): R
  • Controller (PS5/USB): Left stick steer, R2 push, X jump, Square spin
  • Controller extras: Triangle enter event, Options return to hub, L1/R1 switch ride
